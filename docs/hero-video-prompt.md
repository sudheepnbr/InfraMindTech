# InfraMindTech Hero — 10s Datacenter AI Video Prompt

Use this prompt in **Runway Gen-3**, **Kling AI**, **Pika**, or **Sora**.

## Main prompt (copy/paste)

```
Cinematic 10-second corporate datacenter video, slow smooth camera dolly forward through a modern enterprise data center aisle. Rows of black server racks with blue and cyan LED status lights blinking. Close-up passes: enterprise firewall appliance with security shield hologram, core network switches with fiber optic cables and active ports, blade servers with cooling fans. Subtle floating AI data particles and network topology lines in blue (#2563EB) and cyan (#06B6D4). Clean, futuristic, high-tech, photorealistic, shallow depth of field, soft volumetric lighting, no people, no logos, no text. Professional IT infrastructure mood — intelligent, secure, scalable. 4K, 16:9, seamless loop-friendly ending.
```

## Settings

| Setting | Value |
|---------|-------|
| Duration | 10 seconds |
| Aspect ratio | 16:9 |
| Style | Cinematic / Photorealistic |
| Camera | Slow forward dolly, slight parallax |
| Loop | Enable loop (for homepage background) |

## Scene breakdown (optional per-shot)

1. **0–3s** — Wide aisle, server racks both sides, blue ambient light
2. **3–6s** — Rack close-up, blinking LEDs, network cables
3. **6–8s** — Firewall + core switch panel, data flow animation
4. **8–10s** — Pull back to aisle, AI particle overlay, fade to loop point

## After you generate the video

1. Export as **MP4** (H.264), max **10 MB** if possible
2. Save as `images/hero-datacenter.mp4`
3. In admin → **Media Library** → upload the video
4. Set **Hero Video URL** to `/images/hero-datacenter.mp4` (or upload path)
5. Click **Save All**

The homepage hero will play it automatically as a subtle background loop.
